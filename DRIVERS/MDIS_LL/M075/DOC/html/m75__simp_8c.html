Detailed Description

Simple example program for the M75 driver.

Author:
Christian.Schuster@men.de
Date
2009/07/14 17:47:18
Revision
1.4

The program sends and receives one frame on the same channel using the local loopback mode of the controller. For test to make sense, Rx clock must match the Tx clock. Since internally no working solution can be found, the Tx clock is echoed on the TRxC pin and the two clocks have to be connected together:

  • Pin13(DSR+) to Pin14(DTR+) and Pin6(DSR-) to Pin7(DTR-) of corresponding port
  • or Pin9 - Pin10 and Pin19 - Pin20 on the 24 Pin rear I/O receptacle connector

Required: libraries: mdis_api
